# 홍제동 아르바이트 매뉴얼

아르바이트생을 위한 업무 가이드 페이지입니다.

## 목차

- **오픈** — 출근 직후 절차 (무인 해제, 불 켜기, 열쇠 관리)
- **정산** — 1·2·3차 정산 방법 및 근무교대 절차
- **청소** — 오픈 청소 6단계 (흡연실, 화장실, 라인, 계단)
- **요리** — 메뉴별 조리법 및 음료 제조
- **오픈/마감 체크리스트** — 순서대로 따라하는 체크리스트
- **기타** — 행주 구분, 배송 처리 등

## 사용 방법

`홍제동_아르바이트_매뉴얼.html` 파일을 브라우저로 열면 됩니다.

> 일부 민감정보는 `***`로 처리되어 있습니다.
